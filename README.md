# Service Booking Layout

### Features
1.  **Navigation Bar**: Left-aligned logo, center links, and a red Logout button on the right.
2.  **Dynamic Cart**: The left section updates automatically when you click "Add Item" in the right section.
3.  **Booking Form**: A simple form in the left sidebar for user details.
4.  **Service Display**: Service cards with images, prices, and buttons.

### Technologies Used
- HTML5 / CSS3 (Grid & Flexbox)
- JavaScript (DOM manipulation)

### How to use
- Open `index.html`.
- Click "Add Item" on any service card.
- Watch the "Added Items" list and "Total Amount" update in real-time.
